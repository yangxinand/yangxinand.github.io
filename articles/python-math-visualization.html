用 Python 可视化数学函数

2026-03-15 · 编程可视化

数据可视化是理解数学函数最直观的方式。Python 提供了强大的工具来绘制函数图像，本文介绍两种主流方案。

1. Matplotlib 基础绘图

matplotlib 是 Python 最基础的绘图库，适合绘制各类函数图像：

import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-5, 5, 200)
y = x**2
plt.plot(x, y)
plt.title('y = x²')
plt.grid(True)
plt.show()

2. Plotly 交互式绘图

plotly 支持交互式图表，学生可以缩放、平移、查看坐标值，学习体验更好。

3. 教学应用

  • 对比不同幂函数的图像特征
  • 动态展示参数变化对图像的影响
  • 叠加绘制函数与其导函数

编程可视化让抽象的数学概念变得触手可及，是数据教育的有力工具。
